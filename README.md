Romuald Dev Portfolio

📌 Présentation du projet

Romuald Dev Portfolio est un site portfolio personnel développé pour présenter mes compétences, mes projets et mon parcours en développement web.

L'objectif est de créer une interface moderne, responsive et accessible permettant aux visiteurs de découvrir mon profil de développeur web junior.

---

🚀 Fonctionnalités

- ✅ Design moderne en mode sombre
- ✅ Responsive design (ordinateur, tablette et mobile)
- ✅ Menu de navigation mobile avec hamburger
- ✅ Animation d'apparition au scroll
- ✅ Effet machine à écrire sur la section d'accueil
- ✅ Navigation avec lien actif selon la position de défilement
- ✅ Section compétences avec barres de progression
- ✅ Présentation des projets
- ✅ Formulaire de contact avec validation JavaScript
- ✅ Bouton retour en haut de page

---

🛠️ Technologies utilisées

Front-end

- HTML5 : structure et sémantique du site
- CSS3 : design, animations et responsive design
- JavaScript : interactions et fonctionnalités dynamiques

Outils

- Git / GitHub
- Figma (conception UI/UX)
- Visual Studio Code
